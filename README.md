# GitHub Copilot for Non-Developers Workshop

Welcome! This repository contains demo materials for learning how GitHub Copilot can help non-developers be more productive.

## What's Inside

### Demo Files
Real-world scenarios you can practice with:

1. **demo-01-messy-meeting-notes.md** - Transform rough meeting notes into professional format
2. **demo-02-bullet-points-to-format.md** - Expand simple bullets into comprehensive documentation
3. **demo-03-email-draft-rough.md** - Polish casual drafts into professional communications
4. **demo-04-process-documentation-outline.md** - Turn outlines into detailed guides

### Templates
Ready-to-use templates for common tasks:
- **meeting-notes-template.md** - Structure for any meeting
- **project-brief-template.md** - Comprehensive project documentation

### Workshop Guide
**demo-05-workshop-guide.md** - Complete presenter guide with timing and talking points

## How to Use These Materials

### For Attendees:
1. Open this repository in VS Code
2. Ensure GitHub Copilot extensions are installed
3. Open any demo file
4. Try asking Copilot to improve, format, or expand the content
5. Experiment with different prompts!

### For Presenters:
- Follow the workshop guide for a structured 60-minute session
- Each demo includes timing, steps, and Key Points
- Feel free to adapt to your audience

## Example Prompts to Try

With any demo file:
- "Format this professionally"
- "Make this more concise"
- "Expand this with more details"
- "Improve the tone to be more [formal/friendly/urgent]"
- "Convert this to a table"
- "Add a summary at the top"
